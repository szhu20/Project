Statement of Burnside's Lemma

Group theory, a branch of algebra, analyzes situations like rotations of the wheel very closely. In terms of group-theoretic notation, Burnside's Lemma asserts the following:

$$|X/G|=\frac{1}{|G|}\sum_{g\in G}|X^g|.$$

What does all this nonsense mean? It helps to first define the notion of a group. In mathematical terms, a group is a collection of objects $G$, together with an associative binary operation $\cdot$ called multiplication, such that the following properties hold:

  1. (Closure) For all $a,b\in G$, $a\cdot b\in G$. From now on, we will abbreviate $a\cdot b$ as $ab$.
  2. (Identity) There is a unique element $e\in G$ such that $ae=ea=a$ for all $a\in G$.
  3. (Inverse) For each $a\in G$, there is a unique $a^{-1}\in G$ such that $aa^{-1}=a^{-1}a=e$.

It is important to notice that $ab\neq ba$ in an arbitrary group! Commutativity is not a requirement of a group, and we will not assume this is true. A group that is commutative is called Abelian, after Niels Henrik Abel.

What use, then, could a group possibly be in the context of rotating wheels? It turns out that groups are useful in a way called a group action. A group action of a group $G$ on a set $X$ is a function $f: G\times X\rightarrow X$ such that:

  1. (Identity) $f(e,x)=x$, and
  2. (Associativity) $f(gh,x)=f(g,f(h,x))$.

We generally abbreviate $f(a,x)$ as simply $ax$. This is a slight abuse of notation, as $x$ is not a member of $G$; nevertheless, this is the common way of writing group actions. Under this new abbreviation, the previous two axioms become $ex=x$ and $(gh)x=g(hx)$, which suddenly seem like very reasonable axioms.

So we have this definition. What, then, does this this entail? Seeing that a group action is basically a way of assigning each element in a set to another element based on a given element of a group, we can essentially think of the elements of the group as functions. Using this notion, we can easily define some more notation. Define the orbit of an element $x$ with respect to a corresponding group action as the set of elements in $X$ to which $x$ can be mapped to through the group action. Denote this orbit by $\text{Orb}(x)$. Note that if $y\in\text{Orb}(x)$, then $\text{Orb}(y)=\text{Orb}(x)$. To see why this is true, consider an element of $\text{Orb}(y)$. This can be expressed in the form $gy$ by definition. However, since $y\in\text{Orb}(x)$, we know that $y=hx$. Therefore $gy=ghx$, from which we can conclude that $\text{Orb}(y)\subseteq\text{Orb}(x)$. Similarly, suppose that an element in $\text{Orb}(x)$ is $gx$. Observe now that $y=hx\implies x=h^{-1}y$. Therefore we see that $gx=gh^{-1}y$, from which we can conclude that $\text{Orb}(x)\subseteq\text{Orb}(y)$. Thus $\text{Orb}(y)=\text{Orb}(x)$.

This observation allows us to partition $X$ into distinct orbits, where each set is an orbit of the group action. Denote by $X/G$ the set of all orbits of the group action of $G$ on $X$. This leads us to the first major conclusion: $|X/G|$ represents the number of "distinct" elements of $X$, under the group action of $G$. In the case of our wheels, the set $X$ represents all possible wheels we can create using the given colors regardless of rotation, the group $G$ represents the rotations on the wheel, and $|X/G|$ represents the number of different wheels possible when we take rotations into account.

Let us make one more definition: denote by $X^g$ the set of elements of $X$ fixed by $g\in G$. Equivalently, $X^g=\{x\in X\mid gx=x\}$. We then have a (more or less) complete mathematical understanding of Burnside's Lemma. Let's see how this can be applied to the problem at hand.
