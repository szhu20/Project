Introduction to Burnside's Lemma

Consider a ring with eight circles.

Suppose that we can color each circle in one of four colors: red, blue, yellow, and green.

It would seem like there are $8^4=4096$ possible rings we could make from these colors, and this is true if the ring is bolted to the ground. However, what if we allowed ourselves to rotate the ring? There would no longer be $4096$ possible rings, as
